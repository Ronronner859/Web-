乌鸡汤

乌鸡汤属于粤菜，有着悠久的历史，以特有的菜式和韵味，独树一帜，粤菜是中国著名八大菜系之一，在国内外享有盛誉。粤菜的主要性格为:取之自然，烹之自由 ，食之自在!

粤菜广义上来说由广州菜(亦称广府菜)、潮州菜(亦称"潮汕菜")、东江菜(属客家菜)组成，以广州菜作为代表。
